# 테미(Temi) 로봇 프로젝트

테미 로봇을 활용한 웹 기반 제어 및 관리 시스템

## 🍽️ 식당 자리 안내 시스템

테미 로봇을 활용한 식당 자리 안내 시스템입니다. 손님이 입장하면 인원 수를 입력하고, 자동으로 적절한 테이블을 배정하여 테미 로봇이 안내합니다.

### 주요 기능

- 👥 손님 인원 수 입력 및 자동 테이블 배정
- 🗺️ 테이블 현황 실시간 표시 (4개 테이블)
- 🤖 테미 로봇 자동 안내 (음성 + 이동)
- 📊 테이블 사용 상태 관리
- 🎯 최적 테이블 배정 알고리즘

## 🤖 테미 로봇 기본 설정

### 1단계: 하드웨어 설정

#### 전원 및 부팅
1. **전원 연결**: 테미 로봇의 전원 어댑터를 연결합니다
2. **전원 버튼**: 로봇 측면 또는 상단의 전원 버튼을 눌러 부팅합니다
3. **부팅 대기**: 초기 부팅에는 약 1-2분이 소요될 수 있습니다

#### 네트워크 연결
1. **Wi-Fi 설정 접근**: 테미의 터치스크린에서 설정(Settings) 메뉴로 이동
2. **네트워크 선택**: 사용 가능한 Wi-Fi 네트워크 목록에서 연결할 네트워크 선택
3. **비밀번호 입력**: 네트워크 비밀번호를 입력하여 연결
4. **연결 확인**: 인터넷 연결 상태를 확인합니다

> ⚠️ **중요**: 안정적인 인터넷 연결이 필수입니다. 테미의 모든 기능이 네트워크를 통해 작동합니다.

### 2단계: 테미 앱 설치 및 연동

#### 스마트폰 앱 설치
1. **앱 다운로드**: 
   - iOS: App Store에서 "temi" 검색
   - Android: Google Play Store에서 "temi" 검색
2. **앱 설치**: 다운로드 후 설치 완료

#### 계정 생성 및 로그인
1. **앱 실행**: 설치한 테미 앱을 실행
2. **계정 생성**: 새 계정을 생성하거나 기존 계정으로 로그인
3. **권한 허용**: 필요한 권한(위치, 카메라, 마이크 등) 허용

#### 테미와 페어링
1. **로봇 검색**: 앱에서 "로봇 추가" 또는 "페어링" 메뉴 선택
2. **QR 코드 스캔**: 테미 화면에 표시된 QR 코드를 스마트폰으로 스캔
   - 또는 로봇의 고유 ID를 직접 입력
3. **연결 확인**: 페어링이 완료되면 앱에서 테미를 제어할 수 있습니다

### 3단계: 맵핑 및 자율주행 설정

#### 공간 맵핑 (Mapping)
1. **맵핑 모드 시작**: 테미 앱 또는 로봇 화면에서 "맵핑 시작" 선택
2. **로봇 이동**: 테미를 수동으로 이동시키거나 자율주행으로 공간을 스캔
3. **전체 공간 커버**: 모든 방과 복도를 포함하여 전체 공간을 스캔
4. **맵 저장**: 맵핑이 완료되면 맵을 저장하고 이름을 지정

#### 위치 지정 (Waypoints)
1. **위치 추가**: 맵 상에서 특정 위치를 선택
2. **위치 이름 지정**: 각 위치에 이름을 부여 (예: "접수처", "대기실", "회의실")
3. **위치 저장**: 저장된 위치로 테미가 자율주행할 수 있습니다

### 4단계: 음성 및 AI 기능 설정

#### 음성 인식 설정
1. **설정 메뉴**: 테미 화면에서 설정(Settings) → 음성(Voice) 메뉴로 이동
2. **음성 인식 활성화**: 음성 인식 기능을 켭니다
3. **음성 학습**: 테미가 사용자의 음성을 인식하도록 학습시킵니다

#### Google Assistant 연동 (선택사항)
1. **설정 앱 실행**: 테미의 설정 앱에서 "Google" 선택
2. **Google Assistant 설정**: "Google 어시스턴트" → "Hey Google 및 Voice Match" 선택
3. **음성 등록**: "Hey Google" 기능을 켜고 음성 인식 등록 진행
4. **테스트**: "Hey Google" 명령으로 기능 테스트

### 5단계: 소프트웨어 업데이트

1. **업데이트 확인**: 설정 → 시스템 정보 → 소프트웨어 업데이트
2. **업데이트 다운로드**: 업데이트가 있으면 다운로드
3. **재부팅**: 업데이트 후 로봇 재부팅

## 💻 개발 환경 설정

### 필수 요구사항

- **Node.js**: v18 이상
- **npm** 또는 **yarn**: 패키지 관리자
- **Git**: 버전 관리

### 프로젝트 초기화

```bash
# 프로젝트 클론 또는 초기화
git clone <repository-url>
cd temi

# 의존성 설치
npm install
# 또는
yarn install

# 개발 서버 실행
npm run dev
# 또는
yarn dev
```

### 환경 변수 설정

`.env.local` 파일을 생성하고 다음 변수들을 설정합니다:

```env
# 테미 로봇 API 설정
TEMI_API_URL=https://api.temi.com
TEMI_API_KEY=your_api_key_here
TEMI_ROBOT_ID=your_robot_id_here

# 데이터베이스 설정 (필요시)
DATABASE_URL=your_database_url

# 인증 설정
NEXTAUTH_SECRET=your_secret_key
NEXTAUTH_URL=http://localhost:3000
```

## 🔌 API 연동 가이드

### 테미 API 클라이언트 설정

테미 로봇과 통신하기 위한 API 클라이언트를 설정합니다.

#### 기본 API 엔드포인트

테미 로봇은 다음과 같은 방식으로 제어할 수 있습니다:

1. **REST API**: 웹 기반 HTTP API
2. **WebSocket**: 실시간 양방향 통신
3. **Android SDK**: 안드로이드 앱 개발용

#### API 인증

대부분의 테미 API는 인증이 필요합니다:

```typescript
// API 키를 헤더에 포함
headers: {
  'Authorization': `Bearer ${API_KEY}`,
  'Content-Type': 'application/json'
}
```

#### 주요 API 기능

- **로봇 상태 조회**: 현재 위치, 배터리 상태, 네트워크 상태
- **이동 제어**: 특정 위치로 이동, 정지, 회전
- **음성 제어**: TTS (Text-to-Speech), 음성 명령
- **비디오 스트리밍**: 카메라 영상 실시간 스트리밍
- **맵 관리**: 맵 정보 조회, 위치 관리
