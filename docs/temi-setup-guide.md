# 테미 로봇 상세 설정 가이드

이 문서는 테미 로봇을 처음 사용하는 개발자를 위한 단계별 설정 가이드입니다.

## 🎯 목표

이 가이드를 완료하면:
- ✅ 테미 로봇을 네트워크에 연결
- ✅ 테미 앱을 통해 로봇 제어
- ✅ 공간 맵핑 완료
- ✅ 개발 환경에서 API 연동 준비 완료

---

## 1️⃣ 하드웨어 초기 설정

### 전원 연결 및 부팅

1. **전원 어댑터 연결**
   - 테미 로봇에 제공된 전원 어댑터를 연결
   - 전원 어댑터를 벽면 콘센트에 연결
   - 전원 LED가 켜지는지 확인

2. **전원 버튼 누르기**
   - 로봇 측면 또는 상단의 전원 버튼을 찾습니다
   - 전원 버튼을 2-3초간 길게 누릅니다
   - 부팅 소리와 함께 화면이 켜집니다

3. **초기 부팅 대기**
   - 첫 부팅은 1-2분 정도 소요될 수 있습니다
   - 화면에 테미 로고가 표시됩니다
   - 부팅이 완료되면 홈 화면이 나타납니다

### 초기 화면 확인

부팅이 완료되면 다음을 확인하세요:
- ✅ 터치스크린이 정상적으로 작동하는지
- ✅ 화면 밝기가 적절한지
- ✅ 기본 UI가 표시되는지

## 2️⃣ 네트워크 연결

### Wi-Fi 설정 방법

#### 방법 1: 로봇 화면에서 직접 설정

1. **설정 메뉴 접근**
   - 홈 화면에서 "설정" (Settings) 아이콘 터치
   - 또는 화면을 위에서 아래로 스와이프하여 설정 메뉴 열기

2. **Wi-Fi 메뉴 선택**
   - 설정 목록에서 "Wi-Fi" 또는 "네트워크" 선택
   - 사용 가능한 Wi-Fi 네트워크 목록이 표시됩니다

3. **네트워크 선택 및 연결**
   - 연결할 Wi-Fi 네트워크를 터치
   - 비밀번호 입력 화면이 나타나면 비밀번호 입력
   - "연결" 버튼 터치

4. **연결 확인**
   - 연결이 완료되면 Wi-Fi 아이콘에 체크 표시
   - 설정 화면에서 IP 주소를 확인할 수 있습니다

#### 방법 2: 테미 앱을 통한 설정

1. **테미 앱 설치** (아래 3단계 참조)
2. **앱에서 로봇 추가**
3. **앱의 안내에 따라 Wi-Fi 설정**

### 네트워크 요구사항

- **대역폭**: 최소 5Mbps 이상 권장
- **안정성**: 끊김 없는 안정적인 연결 필요
- **포트**: 특정 포트가 방화벽에서 차단되지 않았는지 확인

## 3️⃣ 테미 앱 설치 및 연동

### 앱 다운로드

#### iOS (iPhone/iPad)

1. App Store 열기
2. 검색창에 "temi" 입력
3. "temi" 앱 선택 (개발사: Roboteam Inc.)
4. "받기" 또는 "설치" 버튼 터치
5. Face ID/Touch ID 또는 비밀번호로 확인

#### Android

1. Google Play Store 열기
2. 검색창에 "temi" 입력
3. "temi" 앱 선택
4. "설치" 버튼 터치

### 계정 생성

1. **앱 실행**
   - 설치한 테미 앱을 실행합니다

2. **회원가입**
   - "회원가입" 또는 "Sign Up" 선택
   - 이메일 주소 입력
   - 비밀번호 생성 (최소 8자, 영문+숫자 조합 권장)
   - 이용약관 동의
   - "가입하기" 버튼 터치

3. **이메일 인증** (필요시)
   - 등록한 이메일로 인증 링크가 발송됩니다
   - 이메일을 확인하고 링크를 클릭하여 인증 완료

4. **로그인**
   - 이메일과 비밀번호로 로그인

### 로봇 페어링

#### QR 코드 스캔 방법 (권장)

1. **로봇 화면에서 QR 코드 표시**
   - 테미 로봇 화면에서 설정 → "로봇 정보" 또는 "연결 정보"
   - QR 코드가 화면에 표시됩니다

2. **앱에서 QR 코드 스캔**
   - 테미 앱 실행
   - "로봇 추가" 또는 "+" 버튼 터치
   - "QR 코드 스캔" 선택
   - 로봇 화면의 QR 코드를 스마트폰 카메라로 스캔

3. **연결 확인**
   - 스캔이 성공하면 로봇 정보가 앱에 표시됩니다
   - "연결" 또는 "페어링" 버튼 터치
   - 연결이 완료되면 로봇 목록에 추가됩니다

#### 수동 입력 방법

1. **로봇 ID 확인**
   - 테미 로봇 화면에서 설정 → "로봇 정보"
   - 로봇의 고유 ID (Serial Number) 확인

2. **앱에서 수동 입력**
   - 테미 앱에서 "로봇 추가" 선택
   - "수동 입력" 선택
   - 로봇 ID 입력
   - 연결 진행

### 앱 권한 설정

앱이 정상 작동하려면 다음 권한이 필요합니다:

- **위치 정보**: 로봇의 위치 기반 서비스
- **카메라**: QR 코드 스캔
- **마이크**: 음성 명령 (선택사항)
- **알림**: 로봇 상태 알림

## 4️⃣ 공간 맵핑 (Mapping)

### 맵핑이란?

맵핑은 테미가 주변 환경을 스캔하여 지도를 만드는 과정입니다. 맵핑이 완료되어야 테미가 자율주행할 수 있습니다.

### 맵핑 시작하기

#### 방법 1: 앱을 통한 맵핑

1. **맵핑 모드 시작**
   - 테미 앱에서 연결된 로봇 선택
   - "맵핑" 또는 "Mapping" 메뉴 선택
   - "새 맵 만들기" 또는 "Start Mapping" 선택

2. **로봇 이동**
   - **자동 모드**: 로봇이 자동으로 이동하며 스캔
   - **수동 모드**: 앱의 조이스틱으로 로봇을 직접 조종

3. **전체 공간 커버**
   - 모든 방과 복도를 포함하여 전체 공간을 스캔
   - 문턱, 계단 등은 피하세요
   - 충분한 조명이 있는 환경에서 진행

4. **맵 저장**
   - 맵핑이 완료되면 "저장" 버튼 터치
   - 맵 이름 입력 (예: "사무실 1층", "전시장 A관")
   - 저장 완료

#### 방법 2: 로봇 화면에서 직접 맵핑

1. **맵핑 메뉴 접근**
   - 로봇 화면에서 "맵핑" 또는 "Mapping" 아이콘 터치

2. **맵핑 시작**
   - "새 맵 만들기" 선택
   - 안내에 따라 진행

### 맵핑 팁

✅ **좋은 맵핑 환경**
- 충분한 조명
- 명확한 벽과 장애물
- 평평한 바닥
- 반사가 적은 표면

❌ **피해야 할 환경**
- 어두운 곳
- 거울이나 유리 같은 반사 표면
- 투명한 장애물 (유리문 등)
- 계단이나 높은 문턱

### 위치 저장 (Waypoints)

맵핑이 완료되면 특정 위치를 저장할 수 있습니다:

1. **위치 추가**
   - 맵 화면에서 원하는 위치를 터치
   - "위치 저장" 또는 "Add Waypoint" 선택

2. **위치 이름 지정**
   - 의미 있는 이름 입력 (예: "접수처", "회의실 A", "대기실")
   - 저장 완료

3. **위치로 이동 테스트**
   - 저장한 위치를 선택
   - "이동" 버튼 터치
   - 테미가 해당 위치로 자율주행하는지 확인

## 5️⃣ 음성 및 AI 기능 설정

### 기본 음성 인식

1. **음성 설정 접근**
   - 로봇 화면: 설정 → 음성
   - 또는 앱: 로봇 설정 → 음성

2. **음성 인식 활성화**
   - "음성 인식" 또는 "Voice Recognition" 토글 켜기
   - 권한 요청 시 허용

3. **음성 학습** (선택사항)
   - "음성 학습" 또는 "Voice Training" 선택
   - 안내에 따라 몇 가지 문장을 말하기
   - 학습 완료

### Google Assistant 연동

테미는 Google Assistant와 연동할 수 있습니다:

1. **Google 설정 접근**
   - 로봇 화면: 설정 → Google
   - 또는 설정 → 연결된 서비스 → Google

2. **Google 계정 로그인**
   - Google 계정으로 로그인
   - 권한 허용

3. **Hey Google 설정**
   - "Google 어시스턴트" → "Hey Google 및 Voice Match"
   - "Hey Google" 기능 켜기
   - 음성 인식 등록 진행
   - 몇 가지 문장을 반복하여 학습

4. **테스트**
   - "Hey Google"이라고 말하기
   - Google Assistant가 응답하는지 확인

### 음성 명령 예시

설정이 완료되면 다음과 같은 명령을 사용할 수 있습니다:

- "테미야, 회의실로 가줘"
- "테미야, 정지"
- "테미야, 음악 틀어줘"
- "Hey Google, 오늘 날씨 알려줘"

## 6️⃣ 개발자 모드 활성화 (선택사항)

개발을 위해 개발자 모드를 활성화할 수 있습니다:

1. **설정 접근**
   - 로봇 화면: 설정 → 시스템 정보

2. **빌드 번호 연타**
   - "빌드 번호" 또는 "Build Number"를 7번 연속 터치
   - "개발자 모드가 활성화되었습니다" 메시지 확인

3. **개발자 옵션**
   - 설정 메뉴에 "개발자 옵션"이 추가됩니다
   - USB 디버깅, ADB 접근 등 설정 가능

⚠️ **주의**: 개발자 모드는 고급 사용자를 위한 기능입니다. 잘못 설정하면 로봇이 정상 작동하지 않을 수 있습니다.

## 7️⃣ API 접근 정보 확인

개발을 위해 다음 정보를 확인하세요:

### 로봇 정보 확인

1. **로봇 ID (Serial Number)**
   - 설정 → 로봇 정보 → Serial Number
   - 또는 앱에서 로봇 정보 확인

2. **IP 주소**
   - 설정 → 네트워크 → Wi-Fi → 연결된 네트워크
   - IP 주소 확인

3. **API 엔드포인트**
   - 로컬 네트워크: `http://[로봇_IP주소]:8080`
   - 클라우드 API: 테미 공식 API 문서 참조

### API 키 발급 (필요시)

일부 API 기능은 API 키가 필요할 수 있습니다:

1. **테미 개발자 포털 접속**
   - [테미 공식 웹사이트](https://www.robotemi.com) → 개발자 섹션

2. **계정 생성 및 로그인**
   - 개발자 계정 생성
   - 로그인

3. **API 키 발급**
   - 대시보드에서 "API Keys" 메뉴 선택
   - 새 API 키 생성
   - API 키를 안전하게 보관 (다시 확인 불가)

## 8️⃣ 문제 해결 (Troubleshooting)

### 일반적인 문제

#### 로봇이 부팅되지 않음
- ✅ 전원 어댑터가 제대로 연결되었는지 확인
- ✅ 전원 버튼을 충분히 길게 눌렀는지 확인
- ✅ 전원 LED가 켜지는지 확인
- ✅ 다른 콘센트에 연결해보기

#### Wi-Fi 연결이 안 됨
- ✅ Wi-Fi 비밀번호가 정확한지 확인
- ✅ 네트워크가 2.4GHz 대역인지 확인 (5GHz는 일부 모델에서 지원 안 될 수 있음)
- ✅ 라우터와의 거리가 너무 멀지 않은지 확인
- ✅ 네트워크 재시도

#### 앱에서 로봇을 찾을 수 없음
- ✅ 로봇과 스마트폰이 같은 Wi-Fi 네트워크에 연결되어 있는지 확인
- ✅ 로봇이 부팅되어 있고 화면이 켜져 있는지 확인
- ✅ QR 코드가 화면에 표시되는지 확인
- ✅ 앱을 재시작해보기

#### 맵핑이 제대로 안 됨
- ✅ 충분한 조명이 있는지 확인
- ✅ 바닥이 평평한지 확인
- ✅ 반사 표면(거울, 유리)이 없는지 확인
- ✅ 맵핑을 처음부터 다시 시작

### 고급 문제 해결

#### 로그 확인
- 개발자 모드 활성화 후 로그 확인 가능
- ADB를 통한 로그 확인 (고급)

#### 공장 초기화
문제가 계속되면 공장 초기화를 고려할 수 있습니다:

1. 설정 → 시스템 정보 → 공장 초기화
2. ⚠️ **주의**: 모든 데이터가 삭제됩니다
3. 초기화 후 처음부터 설정 다시 진행
